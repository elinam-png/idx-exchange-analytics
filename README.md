# idx-exchange-analytics
MLS Housing Market Analytics Internship Project

Project Overview

This repository contains my individual work for the IDX Exchange MLS Analytics Internship Program.

The project focuses on analyzing MLS housing market data using Python, Pandas, and Tableau to generate market intelligence, identify trends, and create interactive dashboards.

Project Objectives
Aggregate MLS listings and sold transaction data
Perform exploratory data analysis and validation
Clean and prepare housing market datasets
Engineer market metrics and features
Detect and flag outliers
Develop Tableau dashboards
Produce market intelligence reports
Repository Structure
scripts/ – Python scripts used for data processing and analysis
notebooks/ – Exploratory analysis and testing notebooks
docs/ – Weekly notes and project documentation
outputs/ – Non-confidential outputs, summaries, and visualizations
Workflow
Update listings and sold datasets using Python scripts
Clean and validate data
Engineer market metrics
Analyze trends and market performance
Build Tableau dashboards
Publish final dashboards through Tableau Public
Data Privacy

MLS source data is confidential and is not included in this repository. Only code, documentation, and non-sensitive outputs are stored here.
